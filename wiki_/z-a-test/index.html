    z-a-test

    Introduction#

    Zinit extension that runs tests (via make test, for example) – if it finds any of them – after installing and updating a plugin or snippet. Simply load it as a plugin to make it active:

    zinit light z-shell/z-a-test

    Installation#

    Simply load as a plugin. This will install the extension within Zinit:

    zinit light z-shell/z-a-test
